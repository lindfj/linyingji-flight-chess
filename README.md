# YJ飞行器

竖屏手游风格的四人飞行棋网页。

## 功能

- 联网好友房：创建房间、加入房间、跨设备实时同步
- 四人同屏轮流游戏
- 可修改玩家昵称
- 聊天与快捷消息
- 游戏战况记录
- 同一浏览器多标签页实时同步
- 手机与电脑自适应
- 52 格标准环形赛道
- 左上蓝、右上绿、左下黄、右下红四角机场
- 红右、蓝左、黄下、绿上四色中心终点通道
- 三角起飞安全格与彩色虚线双向飞行通道
- 自动掷骰决定先手
- 掷 6 起飞与连续行动
- 三连 6 惩罚
- 安全起飞格
- 撞子、叠子阻挡与反弹
- 同色跳跃
- 虚线飞行
- 终点通道精确抵达与反弹

## 运行

直接打开 `index.html` 即可游玩。

## 启用联网

1. 在 Supabase 创建免费项目。
2. 打开 SQL Editor，运行 `supabase-schema.sql`。
3. 在 Project Settings -> API 复制 `Project URL` 和 `anon public key`。
4. 填入 `multiplayer-config.js`。
5. 重新上传 GitHub Pages 文件。
